* 使用判断

- =package.el=、=use-package=、Org、project、状态模块和框架内核属于必需能力。
- 第三方主题、Dashboard、Modeline、补全、Magit、Org 美化和 gptel 都有明确配置。
- 缺少第三方包时基础 Emacs 可离线启动；Doctor 会把缺包列为错误，不能据此声称
  完整 Loomacs 已启用。
- AI 凭据和 Language Server 是本机能力，不属于 Emacs 包，也不进入包锁。
