#+title: Loomacs Roadmap
#+author: SSW

* 0.1.0 — 当前落地范围

- [X] =loomacs-= 公共命名空间与作者归属。
- [X] Profile、领域总 Org、插件/能力子 Org。
- [X] 52 单元 Manifest、稳定拓扑排序和显式 staging Tangle。
- [X] generated last-known-good、current/previous Release 与回滚。
- [X] XDG state/cache 边界和旧状态无覆盖迁移。
- [X] macOS frame、字体、默认英文输入源与快捷键边界。
- [X] Batppuccin、Dashboard、Doom Modeline。
- [X] Magit + Transient Posframe。
- [X] Org Modern、Org Appear、Agenda、Capture、源码块。
- [X] Vertico/Orderless/Marginalia/Consult/Embark/Corfu/Cape。
- [X] Eglot/xref/Flymake/Tree-sitter。
- [X] gptel + auth-source 安全默认值。
- [X] 包版本/commit 锁、Doctor、ERT、编译与离线启动检查。

* 0.1.x — 使用反馈后的稳定化

- [ ] 根据真实日常使用调整补全候选数量、Consult 预览延迟和 Corfu 触发策略。
- [ ] 根据实际项目语言决定是否增加 Rust、Go 等 server，不能为“看起来完整”安装。
- [ ] 对 Dashboard、Posframe、Corfu 和 Org 做多显示器/Retina GUI 回归截图。
- [ ] 如作者明确选择许可证，再增加 =LICENSE= 和对外贡献说明。

* 0.2.0 — 多机复现

- [ ] 在第二台 macOS 机器从干净 clone 验证 bootstrap、离线启动和回滚。
- [ ] 评估 =package-vc= 精确 revision 安装；解决同仓库多包等传递依赖边界后再切换。
- [ ] 增加经过验证的额外 Profile；没有真实测试前不宣称 Linux/Windows 支持。

* 非目标

- 不复制 Doom/Spacemacs 框架。
- 不在启动时自动 Tangle、联网、升级或安装。
- 不做隐式 AI 上下文收集或默认工具调用。
- 不为模块数量而继续拆分没有独立职责的文件。
- 第一版不做热重载；正常重启是更明确的发布边界。
